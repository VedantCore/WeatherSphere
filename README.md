WeatherSphere is a modern full-stack Node.js application that transforms raw meteorological data into a stunning, interactive dashboard. It features a cinematic 3D digital globe, smart autocomplete search, and dynamic visualization of live weather conditions for any city on Earth.

## üöÄ Key Features

* **Cinematic UI:** A landing page featuring a rotating 3D digital globe with live data point animations.
* **Smart Autocomplete:** Type-ahead search suggestions powered by the OpenWeatherMap Geo API to distinguish between cities (e.g., Paris, TX vs. Paris, France).
* **Deep Data:** Displays real-time Temperature, "RealFeel", Humidity, Wind Speed, Visibility, and Pressure.
* **Dynamic Backgrounds:** The interface adapts responsively to mobile and desktop screens.
* **Robust Backend:** Built with Node.js and Express, securely handling API requests and environment variables.
